# ☁️ Stellar Split: A Web3 Bill Splitting Calculator

- Wallet Setup
- Wallet Connection
- Balance Handling
- Transaction Flow
- Development Standards

## 🚀 Setup Instructions

2. npm install
3. npm run dev
